# Changelog — Mums'Home POS

Toutes les modifications notables sont documentées ici.
Format basé sur [Keep a Changelog](https://keepachangelog.com/fr/1.1.0/).

## [1.0.1] — 2026-05-05

### Ajouté
- Amélioration de la sécurité des mots de passe (hachage bcrypt)
- Système de "Soft Delete" pour les produits et clients
- Optimisation du build Docker pour la production
- Scripts d'installation et de mise à jour hors-ligne

## [1.0.0] — 2026-05-05

### Ajouté
- Première version de production
- Système de caisse (POS) complet
- Gestion des produits et catégories
- Panier avec pavé numérique tactile
- Gestion des paiements (espèces, carte)
- Calcul automatique du rendu de monnaie
- Agenda et rendez-vous
- Authentification admin
- Base de données SQLite locale
- Déploiement Docker avec scripts d'installation
